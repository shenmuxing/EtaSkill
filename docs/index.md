# MetaSkill Documentation

MetaSkill is a public repository for reusable AI-agent skill management workflows and selected public-safe skills. It focuses on making skill work reproducible: clear contracts, repeatable checks, privacy-safe examples, and documentation that another maintainer can follow without local context.

Use these docs to create, audit, standardize, test, and publish reusable skills.

## Start Here

- New to the repository: read [Getting Started](getting-started.md).
- Converting a private workflow: read [Skill Lifecycle](skill-lifecycle.md).
- Publishing or promoting a skill: read [Public-Safety Policy](public-safety.md) and [Publication Checklist](reference/publication-checklist.md).
- Looking for a skill: open the [Skill Catalog](skills.md).

# FAQ

## Do I Need A Custom Domain?

No. Read the Docs can host the site under a project subdomain such as `project-name.readthedocs.io`. A custom domain can be added later when the project needs a stronger public brand.

## Why Use Sphinx?

Sphinx provides a stable documentation system with navigation, search, cross-references, and hosted builds through Read the Docs. MyST-Parser lets the repository keep writing pages in Markdown.

## Why Keep README Short?

The README is the project entrypoint. Long procedures belong in the documentation site or the relevant skill's `SKILL.md`.

## Can MetaSkill Support Other Agents?

The current repository targets Codex skills first. Other agent formats can be considered later through explicit adapters and validation rules.
